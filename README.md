## About Project

AspiraNila adalah aplikasi mobile yang dirancang sebagai wadah penyampaian aspirasi dan diskusi bagi seluruh civitas akademika Universitas Lampung, meliputi mahasiswa, dosen, dan tenaga kependidikan (tendik).

Melalui aplikasi ini, pengguna dapat mengirim aspirasi, melihat koleksi aspirasi, memberikan dukungan, serta berdiskusi melalui fitur thread. Sistem ini bertujuan untuk menciptakan lingkungan kampus yang lebih partisipatif, transparan, dan responsif terhadap masukan dari civitas akademika.

---

## Project Goals

* Menyediakan media aspirasi digital yang terpusat.
* Memfasilitasi diskusi konstruktif melalui thread.
* Meningkatkan partisipasi civitas akademika dalam pengembangan kampus.
* Membantu admin dalam mengelola dan menindaklanjuti aspirasi.
